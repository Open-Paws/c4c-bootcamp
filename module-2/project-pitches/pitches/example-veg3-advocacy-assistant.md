# VEG3 — AI Assistant for Animal Advocates

**Pitched by:** Sam Tucker-Davis
**Category:** Animal Advocacy

## The Problem
Animal advocates face the same questions and counter-arguments thousands of times. Each conversation requires emotional energy and up-to-date knowledge about nutrition, environmental science, and ethics. Many advocates burn out or share outdated information because they can't keep up with the latest research.

## The Tool
An AI chatbot trained on evidence-based advocacy resources that helps advocates prepare for conversations, draft responses to common arguments, and find relevant statistics quickly. Available via Telegram, web, and API for integration into other advocacy tools.

## Who It Helps
Individual advocates preparing for outreach events, advocacy organisations training new volunteers, and anyone who wants evidence-based responses to common questions about veganism and animal rights.

## Tech I Think It Needs
Large language model API (OpenRouter), vector database for advocacy resources, Telegram bot framework, web frontend, system prompts tuned for compassionate communication.

## Why I Care About This
I built an early version of this before ChatGPT existed. The movement needs tools that are specifically designed for advocacy — not general-purpose chatbots that might suggest "ethical meat" as a compromise.
